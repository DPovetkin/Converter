Описание:
В папке с программой присутствует тестовый файл source.bin с двоичными данными, который использовался для отработки.

Конвертирование в csv идет с задержкой  в полсекунды на строку, чтобы сымитировать долгий процесс конвертации.
Для сконвертированных файлов в папке с программой создаются соответствующие директории. 

При посылании команды на конвертирование ответ в виде ссылки на будущий файл приходит сразу, не дожидаясь окончания
конвертации. 

Если перейти по ссылке до окончания конвертации, в ответе будет процент выполнения, если после, то
начнется скачивание. Также процент выполнения придет, если повторно запустить команду на конвертацию того же файла,
не дождавшись окончания конвертирования.В других же случаях повторное конвертирование перезапишет выходной файл. 


ПРИМЕРЫ КОМАНД:
http://localhost:8080/?command=convert&filename=source.bin&type=db (type=csv) - конвертирование в SQLite

http://localhost:8080/?command=download&filename=source.db (.csv)        - скачивание файла

http://localhost:8080/?command=delete&filename=source.db (.csv) - удаление файла

http://localhost:8080/?command=select&filename=source.db&id=25 - выборка одной строки из базы SQLite 

http://localhost:8080/?command=stop - остановка сервера
